# FolioVista Books Security and Operations Case Studies

This repository is a private, sanitized source collection of defensive security and operations case studies maintained for FolioVista Books.

It is designed as source material for practical digital books, field guides, sample chapters, and operational publishing work led by the platform founder.

## About FolioVista Books
- Digital publishing platform for practical guides, field books, manuals, and sample chapters
- Focused on clear, structured, operational learning material
- Built to turn real defensive workflows into clean, export-ready publication assets

## What this repo includes
- Linux VPS hardening workflows (UFW + Fail2ban + SSH safety)
- Network incident troubleshooting playbooks
- Connection diagnosis automation scripts
- Structured case-study documentation and outcomes
