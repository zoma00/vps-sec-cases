# Evidence Guidelines

Use this format in each project README.

## Recommended sections
1. Objective
2. Environment (sanitized)
3. Problem statement
4. Steps executed
5. Validation checks
6. Results and impact
7. Lessons learned

## Evidence quality rules
- Use command snippets with redacted values
- Use before/after outcomes where possible
- Include only safe screenshots with metadata removed
- Prefer reproducible steps over chat transcripts
