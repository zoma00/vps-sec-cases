# Scope and Ethics

## Scope
This portfolio demonstrates defensive IT/security operations work:
- System hardening
- Network troubleshooting
- Service reliability and monitoring
- Incident response playbooks

## Out of scope
- Unauthorized access attempts
- Evasion/anonymity tactics for bypass purposes
- Any activity that violates laws, contracts, or platform policies

## Disclosure model
All examples are sanitized and redacted for public sharing.
Sensitive infrastructure and client data are excluded.
